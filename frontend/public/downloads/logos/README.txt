Mo. — Logo Kit
================

Files included:

  SVG (scalable, best for web/print — uses Google Fonts Gloock):
    mo-logo-dark.svg          ink on transparent
    mo-logo-light.svg         cream on transparent
    mo-logo-on-cream.svg      ink on cream background
    mo-logo-on-ink.svg        cream on ink background

  PNG (rasterised at common sizes):
    mo-logo-dark-512.png      512×512  transparent
    mo-logo-dark-1024.png     1024×1024 transparent
    mo-logo-dark-2048.png     2048×2048 transparent
    mo-logo-light-512.png     512×512  transparent (use on dark bg)
    mo-logo-light-1024.png    1024×1024 transparent (use on dark bg)
    mo-logo-on-cream-1024.png 1024×1024 with cream background
    mo-logo-on-ink-1024.png   1024×1024 with ink background
    mo-wordmark-dark-1600.png 1600×800 horizontal wordmark

  Favicon:
    favicon-32.png            32×32
    favicon-180.png           180×180 (Apple touch icon)

Colors:
  Ink:     #121212
  Cream:   #F3F2ED
  Accent:  #E83B22 (the period)

Typeface:
  Gloock by Google Fonts (https://fonts.google.com/specimen/Gloock)

Usage:
  - SVG is the master. Use SVG wherever you can — web, Figma, print.
  - For PDFs and Office docs, use the 2048 PNG.
  - For social profile photos, use mo-logo-on-ink-1024.png (square w/ bg).
  - Never stretch non-proportionally. Never recolor the accent period.
